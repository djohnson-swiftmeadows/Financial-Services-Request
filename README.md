# Financial Services Request

A modern front-end application for managing and tracking financial service requests.

## Features
- Dashboard view with summary cards
- Search and status filters
- Request list with progress indicators
- Responsive layout styled to match the provided screenshot

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown by Vite.
